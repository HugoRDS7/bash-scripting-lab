## Usage

To run a script:

1. Open a terminal.
2. Navigate to the folder containing the script.
3. Make the script executable (if not already):
4. Run the script:

```bash
chmod +x script_name.sh
./script_name.sh
````
Some scripts may take arguments, e.g.:

```bash
./countdown.sh 5
`````
